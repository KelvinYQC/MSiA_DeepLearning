def decisonBound(model, X, y):
    # Build model
    X = np.array(X)
    y = np.array(y)
    model.fit(X, y)
    # Define a grid
    x_min, x_max = -0.5, 1.5
    y_min, y_max = -0.5, 1.5
    xx, yy = np.meshgrid(np.linspace(x_min, x_max, 100), np.linspace(y_min, y_max, 100))
    X_grid = np.column_stack([xx.ravel(), yy.ravel()])
    # Use the predict method to classify the points in the feature space
    Z = model.predict(X_grid)
    Z = Z.reshape(xx.shape)
    # Create a contour plot of the decision surface
    plt.contourf(xx, yy, Z, cmap=plt.cm.Paired)
    # Plot the input data points on top of the decision surface
    plt.scatter(X[:, 0], X[:, 1], c=y, cmap=plt.cm.Paired, edgecolors='black')
    plt.xlim(x_min, x_max)
    plt.ylim(y_min, y_max)
    plt.show()

decisonBound(SVC(), X, y)
decisonBound(SVC(kernel = "linear"), X, y)
decisonBound(LogisticRegression(), X, y)

The XOR gate can not be fully learned. XOR is not a linearly separable function and linear models are not able to learn a proper decision boundary. We can use kernel trick from SVM to learn the gate; however, logistic regression only produce linear decision boundry. Therefore, logistic regression can not learn XOR gate.
